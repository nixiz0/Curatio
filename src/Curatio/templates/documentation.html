Lines & Columns

Explanation of how the manipulation of rows and columns works in Curatio.

For each button you will need to choose the file you want to import

Columns Buttons

Del-Col allows you to delete columns, after uploading your file on the page you put the name of the column(s) you want to delete (separated by a comma only, no need space between names)

Add-Col allows you to add columns, you put the name of the column(s) you want to add.

Modify-Col allows you to modify column, you put the name of the column you want to modify and on the input below you enter the new column name.


Rows Buttons

Del-Row allows you to delete rows, after uploading your file on the page you put the number of starting, ending interval and this will delete all the rows that are in this range.

Add-Row allows you to add rows, you put the name of the column, the number of starting, ending interval and the value you want to add.

Modify-Row allows you to modify rows of a column, it works the same as adding lines but here the modification will just overwrite the lines to replace with the new value.


Interval Buttons

Interval allows you to put the name of a column and then specify a starting and ending interval, and this will allow you to retrieve only what is in your interval within your rows.

Example : if you put the starting interval 1 and the ending nothing, in a column which in all its lines begins with 'hello' then you will only get 'ello'.


OrderBy Buttons

You must put the name of the column you want to sort and either use the OrderBy button which sorts in ascending order or the ReverseOrder which sorts in descending order.

Missing Values

Explanation of how the manipulation of missing values buttons works in Curatio.

For each button you will need to choose the file you want to import.

Mean Button

You have to put the name of the column which has missing values and it will automatically
replace the missing values with the average.

You must only put the columns which have numerical values

Median Button

You must put the name of the column which has missing values and this will automatically
replace missing values with the median.

You must only put the columns which have numerical values

Standard Deviation Button

You must put the name of the column which has missing values and this will automatically
replace missing values with the standard deviation.

You must only put the columns which have numerical values

Delete Button

You must put the name of the column which has missing values and this will automatically
delete rows who contains missing values.

Define Value Button

You must put the name of the column and enter the value with which you want to
replace the missing value.

Mode Button

You have to put the name of the column which has missing values and it will automatically
replace the missing values with the mode.

You must only put the columns which have categorical values

Transformation Variables

Explanation of how the manipulationof of transformation variables buttons works in Curatio.

For each button you will need to choose the file you want to import

OneHot Button

You have to put the name of the column which has categorical values and it will automatically
encoding by one hot all the categorical values.

You must only put the columns which have categorical values for encoding

Fusion Button

You must put the name of the first column and then the second that you want to merge and finally you
put the name of the column that the merge column will have.

Label Encoder Button

You have to put the name of the column which has categorical values and it will automatically
encoding by label encoding all the categorical values.

Advice : if you have more than 2 different values don't use the label encoder but rather the one hot encoding.

Graphic Tools

Histogram

You can create histograms, in label y there will be the frequency and in x the values of the column you entered.

You can put several numeric column names and this will download a zip file with all your graphics.

Scatter Plot

You can create a scatter plot, you must enter 2 numeric column names (separated by a comma without space) and it will make a scatter plot of your 2 columns, one in x and the other in y.

BoxPlot

You can create a boxplot, you must enter numeric column names you can put one or more column names and it will make a boxplot of each columns, if you put several numeric column names you will have a zip file download with all your boxplots.

Pie

You can create a pie, you can enter one or several column names and it will build you a Pie chart of your columns and their values.

Heatmap

You can create a pie, you must enter several numeric column names and it will build you a Heatmap of correlation between your columns.

Violin

You can create a violin, you can enter one or several numeric column names and it will build you a Violin graph for your columns.

Butterfly

You can create a butterfly graph, you must enter 2 column names and it will build you a Butterfly graph for your 2 columns.

Stem

You can create a stem graph, you must enter 2 column names and it will build you a Stem graph for your 2 columns.

Bar Chart

You can create a bar chart, you can enter one or several column names .
In label y you will have counts and on x your values for your column.
If you put several column names you will have a zip file download with all your bar charts.

Statistic

Statistic Analysis

Describe Statistic

You can see a quick statistical tour of your data.

Graphic Statistic

You can see a some statistic graph of your data.
You can choose the numeric column histogram and boxplot in the drop-down list.

Quartiles

You can see quartiles distribution of your numeric columns.

Mean & Std

You can see mean and standard deviation distribution of your numeric columns.

Min & Max

You can see the min and the max values of your numeric columns.

Statistic Tools

PCA

You can do a PCA of your data.
This normalizes your data and do a Principal Components Analysis.

Important-Features

You can do a important features selection of your data.
It use a RandomForest Features Importantes for the selections and return you a statistic graphic of the most importants features.

Chi-Square

You can do a Chi-Square Test of your data.
Allows you to check the absence of a statistical link between two variables X and Y.

Shapiro-Wilk

You can do a Shapiro-Wilk Test of your data.
Indicates whether a distribution follows a normal distribution or not.

Converter

File-Converter

Each image redirects to an online converter
to convert your files to csv so they can be used
in the Curatio app for processing.

Types-Converter

Each buttons redirects to a types converter
to force the conversion for your columns types.
You have to put the column name that you want to types convert.
Be careful when you want to force the conversion of a column type, this can overwrite data !
Be sure of your conversion !
