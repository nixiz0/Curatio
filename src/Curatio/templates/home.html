WHAT IS CURATIO ?

Curatio is an application to facilitate preparation and processing of data. This application allows you to manage database columns intuitively, by offering the possibility of modifying them, deleting them or creating new ones in just a few clicks.

In addition, it offers functionalities to handle missing values as well as to perform transformations of variables into one-hot or dummy variables (dummies).

In summary, this application aims to make your work easier in order to process your data more quickly and efficiently.

WHY I DID CURATIO?

I mainly designed Curatio with the aim of achieving massive time savings when preparing data for building AI models.

Additionally, to minimize the need to use code to perform data processing and thus make the data processing accessible to as many people as possible in a simple and quick way.
